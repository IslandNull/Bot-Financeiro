# Codex Logic & Implementation Rules

Seu papel é ser o motor de lógica e algoritmos para o Bot Financeiro.

## 🎯 Foco Principal
- Implementação de fórmulas financeiras (Cálculo de juros, saldo, projeções).
- Processamento de strings e dados de entrada do usuário.
- Otimização de loops e manipulação de arrays no Apps Script.

## 🛠 Padrões de Código
- Escrever código limpo e focado em performance.
- Seguir as definições de variáveis estabelecidas no `Code.js`.
- Notificar mudanças estruturais no `AI_WORKFLOW.md`.

## 📂 Integração
- Ler o `GEMINI.md` para entender a arquitetura.
- Seguir o `CLAUDE.md` para formatação de código.
